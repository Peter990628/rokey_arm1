# **💊 약국 업무 보조 로봇 (Pharmacy Assistant Robot)**

**ROKEY 협동로봇 1기 C\_2조 프로젝트**

조제실 내 단순 반복적인 의약품 리필 및 패키징 업무를 로봇이 대신 수행하여, 약사의 업무 피로도를 줄이고 최종 검수에 집중할 수 있는 환경을 구축합니다.

## **📋 프로젝트 개요**

약국의 조제실 내 업무는 오작동 시 리스크가 매우 치명적이므로 인간의 검수가 필수적입니다. 본 프로젝트는 고가의 비전(Vision) 센서 없이 **정밀 좌표 교시(Teaching)와 로봇의 순응 제어(Compliance Control)** 만을 활용하여 조제대 내 의약품 보충과 패키징 작업을 자동화하는 협동로봇 시스템을 구현합니다.

## **✨ 핵심 기능 (Features)**

1. **UI 기반 의약품 잔량 관리 및 사전 보충 (Refill)**  
   * 처방전 DB를 연동하여 특정 알약 소진이 예상될 경우, 조제 전 사전 보충 수행.  
   * 로봇이 서랍형 조제기를 열고, 새 약통을 가져와 정밀한 각도로 기울여 알약을 투하.  
2. **검수 완료 의약품 패키징 (Packaging)**  
   * 약사의 육안 검수가 완료된 투명 약봉지를 그리퍼로 파지하여 종이봉투에 자동 투입.  
3. **UI 상태 알람 및 모니터링 (HMI)**  
   * 웹 기반 UI를 통해 로봇의 현재 상태(대기 중, 제조 중, 완료) 실시간 표시.  
   * 작업 완료 시 DB 업데이트 및 사운드 알람 발생.

## **🛠 시스템 아키텍처**

### **1\. 하드웨어 구성**

* **Robot:** Doosan Robotics M0609 & 그리퍼 툴  
* **Environment:** 조제기 서랍, 약통, 종이봉투 고정 지그, 투명 약 봉투 거치대  
* **Server:** 로봇 제어 및 DB 구동용 Ubuntu 22.04 PC

### **2\. 핵심 제어 기술**

* **좌표 이동 제어:** movej, movel, movejx를 활용한 정밀 이동 및 반복 수행.  
* **순응 제어 (Compliance Control):** task\_compliance\_ctrl을 활용하여 조제기 서랍 파지 및 종이봉투 투입 시 발생할 수 있는 충돌 및 손상 방지.  
* **기울임 제어:** 약이 튀지 않도록 약통 입구를 기준으로 정밀하게 회전(Rx, Ry, Rz)하는 틸팅 로직.

## **📡 ROS2 통신 구조 (Implementation)**

전체 시스템은 역할에 따라 크게 3개의 노드(Node)로 구성되며, Action, Service, Topic을 통해 유기적으로 통신합니다.

### **📦 Nodes**

* task\_manager\_node: 처방전 DB 접근 및 작업 순서 스케줄링 (Main Controller)  
* m0609\_manipulation\_node: 실제 M0609 로봇팔 이동, 서랍 개폐, 정밀 기울임 등 물리적 제어  
* ui\_alarm\_node: 시스템 상태 UI 표시 및 사운드 알람 제어

### **🚀 Actions (로봇 작업 수행)**

* /refill\_process: 약품 가져오기 및 조제기 투하 시퀀스 (Feedback: 현재 진행 단계)  
* /packaging\_process: 투명 약봉투 파지 및 종이봉투 삽입 시퀀스

### **⚙️ Services (상태 동기화)**

* /update\_db\_status: 약품 리필 완료 등 특정 작업 완료 후 DB 서버 동기화  
* /trigger\_sound\_alarm: UI 알람 노드에 단발성 사운드 재생 요청

### **📊 Topics (실시간 모니터링)**

* /robot\_current\_status: M0609 로봇의 현재 작업 상태(IDLE, WORKING 등) 발행  
* /gripper\_torque\_feedback: 동작 중 그리퍼에 가해지는 부하 데이터를 발행하여 충돌 방지
